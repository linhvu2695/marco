# Marco
Project using .NET 7 Web Application for storing data about Countries and Cities around the world. Provide educational quizzes.<br>
### Tech stack:
- .NET API/MVC
- SQL Server
- ElasticSearch
- Redis
- AWS S3
- Prometheus/Grafana
